Missing Key

A game developed during the game makers tool kit game jam 2020 in 48h under the out of control. The game plays with the theme's name, you need to assemble a keyboard, keyboard keys spawn and you need to place them, all the keys can appear but no control keys.

The keys on the in-game keyboard will be set to the position of your own keyboard. If you place the "Y" key where your "J" key is, for example, anytime you press "J" in real life, a "Y" will be written in-game, so keep that in mind.

The team was composed of 4 members one game designer, one artists and one developer. My role was as developer, I programmed all the gameplay for the game, UI, random level generation, music and sounds.

The biggest challenge for this game was to make a system that would switch the key entered to the keyboard to the key setted for the user, so you would use the keyboard you set up, as well as making it work for all keyboard configurations(QWERTY, AZERTY, etc).
